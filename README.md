### NK
> NK 意为nodejs keep
### 支持的功能
+ http/https 服务
    + 权限管理
    + 静态路由
    + 动态路由
    + 挂载路由
    + 数据库
    + 邮件
+ rtmp 服务
+ rtsp 服务
+ tcp 服务
+ udp 服务
### 使用方法
> npm install

> 开发环境 npm run dev

> 生产环境 npm run pro

> 打包发布 npm run build && npm run start

### demo 地址

静态路由带权限：http://127.0.0.1:8081/public/index.html

动态路由：http://127.0.0.1:8081/dynamic/test

挂载路由：http://127.0.0.1:8081/mount/weibos
